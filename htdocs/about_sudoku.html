Sudoku!
A simple yet challenging game where the main objective is to fill the 9x9 board with numbers.
There are a set of rules for normal Classic Sudoku that must be followed for the solution to count.

| | - - - - - - - - - - - - - - - | | - - - - - - - - - - - - - - - | | - - - - - - - - - - - - - - - | |

1. Each of the nine rows have to contain every digit from 1 to 9.
2. Each of the nine columns also have to contain every digit from 1 to 9.
3. In addition, each of the highlighted 3x3 boxes have to contain every digit from 1 to 9.

The final solution must have an equal number of every digit spread evenly across the entire board.
